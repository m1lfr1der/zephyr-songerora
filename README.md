# zephyr-songerora

Immerse your users in a musical experience with Zephyr Songerora, offering music streaming and audio-related features for dApps.

## Installation

```bash
git clone https://github.com/m1lfr1der/zephyr-songerora.git
cd zephyr-songerora
npm install
npm start
```

## Usage
```bash
node index.js
```
